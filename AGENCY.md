# 🏢 AI 개발 에이전시 — 사업 기획서

## 비전
> AI 팀을 활용한 초저가 고품질 웹개발 서비스.
> "누구나 부담 없이 전문가 수준의 결과물을 받을 수 있다."

## 핵심 전략: 박리다매 (High Volume, Low Price)
AI 발전으로 개발 원가가 거의 0에 수렴 → 가격을 파괴적으로 낮추고 물량으로 승부.

### 왜 이게 먹히나?
1. **전통 외주**: 프리랜서 일당 30-80만원 × 2-4주 = 300-1,000만원+
2. **우리**: AI 팀이 1-3일 완성 × 서버비만 → 원가 거의 0
3. **시장**: 소상공인/1인 사업자는 300만원 못 냄. 30-50만원이면 냄.
4. **볼륨**: 300만원짜리 1건 vs 30만원짜리 20건 → 후자가 매출 2배

### 경쟁 우위
| 우리 | 전통 외주 |
|------|----------|
| 1-3일 완성 | 2-4주 |
| 30-100만원 | 300-1,000만원 |
| 24시간 작업 가능 | 근무시간만 |
| 즉시 수정 | 수정 비용 추가 |
| AI 투명 공개 | - |

## 사업 구조

### 팀 편성
```
Luchello (대표)
  ├─ 영업/소통: 클라이언트 미팅, 요구사항 청취, 최종 납품
  ├─ 사업 관리: 수금, 마케팅, 파트너십
  └─ 최종 승인: 납품 전 확인

Luveric (PM / 기술 총괄)
  ├─ 프로젝트 관리: 요구사항 → 태스크 분해 → 에이전트 배정
  ├─ 품질 관리: QA 루프 총괄, 최종 기술 검수
  ├─ 학습/개선: 매 프로젝트 lessons.md 축적
  │
  ├─ 🛠️ Builder (개발팀)
  │    ├─ spawn(opus): 핵심 코딩, 아키텍처
  │    ├─ Claude Code CLI: 복잡한 리팩토링, 디버깅
  │    └─ spawn(기본): 단순 수정, 콘텐츠 작업
  │
  ├─ 🔍 QA (품질팀)  
  │    ├─ spawn: 코드 리뷰, 빌드 검증
  │    ├─ spawn: 접근성/SEO 감사
  │    └─ 체크리스트 기반 검수
  │
  └─ 🎨 Scout (조사/디자인팀)
       ├─ spawn: 시장 조사, 경쟁사 분석
       └─ spawn: 디자인 레퍼런스 수집
```

### AI 사용 투명 정책
- 클라이언트에게 "AI 팀이 개발합니다" 명시
- 이게 오히려 장점: "그래서 이 가격에 이 속도가 가능합니다"
- 최종 품질은 QA 루프로 보장

## 서비스 라인업

### 🥉 Basic (30-50만원)
- 랜딩 페이지 / 포트폴리오 사이트
- 1-5 페이지
- 반응형 (모바일 최적화)
- 기본 SEO
- 납기: 1-2일
- 수정: 2회 포함

### 🥈 Standard (50-100만원)
- 기업 홈페이지 / 쇼핑몰 프론트
- 5-15 페이지
- 반응형 + 애니메이션
- SEO + OG태그 + JSON-LD
- 문의 폼 (이메일 연동)
- 납기: 2-5일
- 수정: 3회 포함

### 🥇 Premium (100-200만원)
- 풀스택 웹앱 / 관리자 패널
- 인증/결제 연동
- DB 설계 + API
- 납기: 5-14일
- 수정: 5회 포함

### ⚡ Express 옵션 (+50%)
- 24시간 이내 납품
- Basic 한정

## 수익 모델

### 월 목표 (Phase 1: 자리잡기)
- Basic × 5건 = 150-250만원
- Standard × 2건 = 100-200만원
- **월 250-450만원** (순이익 90%+, 원가가 서버비뿐)

### 비용 구조
- 서버: ~2만원/월 (t3.small)
- Claude 구독: ~28만원/월 ($200)
- Vercel/호스팅: 무료 티어
- **총 고정비: ~30만원/월**
- ⚠️ 플랫폼 수수료: 크몽 20%, 숨고 ~10-15% (견적에 반영!)
- ⚠️ 세금: 3.3% 원천징수 (프리랜서) 또는 사업자 등록 후 부가세 10%

### Phase 2: 스케일업
- 반복 고객 확보 → 유지보수 월정액 (5-10만원/월)
- 템플릿화: 업종별 사이트 틀 미리 만들어두기
- 자동화: 접수 → 개발 → 납품 파이프라인 최대한 자동

## 마케팅 채널

### 초기 (비용 0)
1. **크몽/숨고/위시켓** — 프리랜서 플랫폼 등록
2. **네이버 블로그** — "AI로 웹사이트 만들기" 콘텐츠 마케팅
3. **인스타그램** — 제작 사례 Before/After 포스팅
4. **지인/건설업 네트워크** — Luchello의 현장 인맥 활용
5. **대성몰탈 사이트** — 실제 포트폴리오로 활용

### 성장기
1. 구글/네이버 광고 (키워드: "저렴한 홈페이지 제작")
2. 제휴: 디자이너, 마케터와 협업
3. 후기/리뷰 마케팅
