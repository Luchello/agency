# 비즈니스 모델

> NexusPoint의 수익 구조, 시장 포지셔닝, 성장 전략을 정리한 문서입니다.

---

## 핵심 전략: 박리다매 (High Volume, Low Price)

AI 발전으로 개발 원가가 거의 0에 수렴 → 가격을 파괴적으로 낮추고 물량으로 승부.

**왜 이게 작동하는가?**

1. **전통 외주**: 프리랜서 일당 30~80만원 × 2~4주 = 300~1,000만원+
2. **우리**: AI 팀이 1~3일 완성 × 서버비만 → 원가 거의 0
3. **시장**: 소상공인/1인 사업자는 300만원 못 낸다. 30~50만원이면 낸다.
4. **볼륨**: 300만원짜리 1건 vs 30만원짜리 20건 → 후자가 매출 2배

### 한국 웹개발 외주 시장 (2025-2026)

| 채널 | 랜딩페이지 | 기업 홈페이지 | 쇼핑몰 |
|------|-----------|-------------|--------|

| 숨고 | 50~150만원 | 150~400만원 | 300~800만원 |
| 위시켓 | 100~300만원 | 300~1,000만원 | 500~2,000만원 |

### NexusPoint 포지셔닝

| 등급 | 가격 | 시장 대비 |
|------|------|----------|
| Basic | 30~50만원 | 크몽 하한가 수준 |
| Standard | 50~100만원 | 크몽 중간 ~ 숨고 하한 |
| Premium | 100~200만원 | 숨고 하한 ~ 위시켓 하한 |

**가격이 낮아도 되는 이유**
- 인건비 0: AI가 개발 → 인건비 없음
- 속도: 1~3일 완성 → 동시에 여러 프로젝트 가능
- 재사용: 업종별 템플릿 축적 → 갈수록 빨라짐

## 성장 전략

### 템플릿화

반복 주문이 많은 업종별로 미리 틀을 만들어두면 개발 시간 대폭 단축.

| 업종 | 현황 |
|------|------|
| 건설/시공업 | 대성몰탈로 경험 보유 |
| 음식점/카페 | 카페 온도로 경험 보유 |
| 뷰티/네일 | 라뮤르 네일로 경험 보유 |
| 의료/치과 | 미소플러스로 경험 보유 |
| 학원/교육 | 모의 테스트 완료 |
| 쇼핑몰 | 향후 개발 |

### 스케일업

- 반복 고객 확보 → 유지보수 월정액 (5~10만원/월)
- 접수 → 개발 → 납품 파이프라인 자동화 (Phase 2)
- 직접 영업 비중 증가로 수수료 절감
