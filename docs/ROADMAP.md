# 로드맵

> NexusPoint의 현재 상태와 단계별 성장 계획입니다.

---

## 현재 상태 (완료)

- 골든 템플릿 구축 완료 (Vite + React + Tailwind CSS v4)
- 자동화 개발 파이프라인 구축 (10건 테스트, 9/10 통과)
- 포트폴리오 사이트 6개 제작 완료
- 크몽 서비스 등록 초안 준비
- 에이전시 홈페이지 배포 (nexuspoint-a.vercel.app)
- QA 체크리스트 프레임워크 확립

---

## Phase 1: 시장 진입 (1~3개월)

**목표**: 크몽 리뷰 5건+ 확보, 월 5~8건 수주

### 핵심 작업

- [ ] 크몽 서비스 정식 등록 (선착순 5건 30만원 프로모션)
- [ ] 숨고 서비스 등록
- [ ] 첫 3건: 20만원 할인 진행 → 리뷰/후기 확보
- [ ] 네이버 블로그 콘텐츠 마케팅 시작
- [ ] 인스타그램 Before/After 포스팅

### 성공 기준

- 크몽 리뷰 5건 이상
- 월 순이익 200만원+
- 재의뢰 고객 1명 이상

---

## Phase 2: 안정화 (3~6개월)

**목표**: 직접 영업 채널 확보, 월 10~15건 수주, 유지보수 계약 확보

### 핵심 작업

- [ ] 직접 영업 채널 구축 (카카오톡 비즈니스 채널)
- [ ] 유지보수 월정액 계약 3건+ (월 5~10만원/건)
- [ ] 업종별 템플릿 완성도 향상 (6개 업종 → 8개)
- [ ] 네이버 광고 소액 테스트 (월 10~20만원)
- [ ] 디자이너/마케터 제휴 파트너 1명

### 성공 기준

- 월 순이익 500만원+
- 유지보수 계약 3건 이상
- 직접 영업 비중 30%+

---

## Phase 3: 스케일업 (6~12개월)

**목표**: 파이프라인 완전 자동화, 월 20건+

### 핵심 작업

- [ ] 접수 폼 → 자동 견적 → 개발 파이프라인 완전 자동화
- [ ] 10개 업종 템플릿 완성
- [ ] 유지보수 대시보드 구축
- [ ] 월 매출 1,000만원+ 달성
- [ ] 사업자 등록 (부가세 체계 전환)

### 성공 기준

- 월 순이익 900만원+
- 템플릿 10종 완성
- 자동화율 80%+

---

## 기술 로드맵

| 항목 | Phase | 설명 |
|------|-------|------|
| QA 자동화 스크립트 | Phase 1 | Lighthouse CLI + axe-core 자동 감사 |
| 접수 폼 | Phase 2 | 웹 기반 프로젝트 의뢰 폼 |
| 자동 견적 시스템 | Phase 2 | 선택지 기반 자동 가격 산출 |
| 클라이언트 대시보드 | Phase 3 | 프로젝트 진행 상황 실시간 확인 |
| NexusPoint 명함 SaaS 통합 | Phase 3 | 홈페이지 에이전시 + 명함 SaaS 브랜드 통합 |

---

## 장기 비전: 브랜드 통합

현재 NexusPoint는 홈페이지 에이전시로 운영 중입니다. 향후 NexusPoint 명함 SaaS와 하나의 브랜드로 통합하여 "AI 기반 비즈니스 디지털화" 종합 서비스로 확장할 계획입니다.

```
NexusPoint
  ├─ 홈페이지 에이전시 (현재)
  │    └─ 소상공인/1인 사업자 대상 웹사이트 제작
  └─ 명함 SaaS (예정)
       └─ 디지털 명함, 링크인바이오 서비스
```
