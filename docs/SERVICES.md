# 서비스 안내

> NexusPoint의 서비스 등급, 가격표, 납품물, 수정 정책, 업종별 특화 정보입니다.

---

## 서비스 등급

### Basic — 30~50만원

**대상**: 랜딩페이지, 포트폴리오 사이트, 원페이지 소개 사이트

| 항목 | 내용 |
|------|------|
| 페이지 수 | 1~5페이지 |
| 반응형 | 모바일 최적화 포함 |
| SEO | 기본 (title, meta description) |
| 수정 | 2회 포함 |
| 납기 | 1~2일 |

**포함 기능**: 반응형 레이아웃, 기본 SEO, Vercel 호스팅

---

### Standard — 50~100만원

**대상**: 기업 홈페이지, 서비스 소개 사이트

| 항목 | 내용 |
|------|------|
| 페이지 수 | 5~15페이지 |
| 반응형 | 반응형 + CSS 애니메이션 |
| SEO | SEO + OG태그 + JSON-LD |
| 폼 | 문의 폼 (이메일 연동) |
| 분석 | Google Analytics |
| 수정 | 3회 포함 |
| 납기 | 2~5일 |

**포함 기능**: Standard 이하 전부 + 문의 폼, 구글 애널리틱스

---

### Premium — 100~200만원

**대상**: 풀스택 웹앱, 관리자 패널, 인증/결제 연동 서비스

| 항목 | 내용 |
|------|------|
| 기술 스택 | Next.js + Supabase |
| 인증 | Supabase Auth / NextAuth |
| 결제 | 토스페이먼츠 / 포트원 |
| DB | PostgreSQL (Supabase) |
| API | Next.js API Routes |
| 수정 | 5회 포함 |
| 납기 | 5~14일 |
| 유지보수 | 1개월 무료 포함 |

**포함 기능**: Premium 이하 전부 + 풀스택, 관리자 패널, 성능 최적화

---

### Express 옵션 — +50%

- Basic 등급 한정
- 24시간 이내 납품 보장
- 예시: Basic 40만원 → Express 60만원

## 수정 정책

- **Basic**: 2회 포함 (납품 후 7일 이내)
- **Standard**: 3회 포함 (납품 후 7일 이내)
- **Premium**: 5회 포함 (납품 후 30일 이내)
- **초과 수정**: 건당 5만원
- **도메인 비용**: 별도 (연 1~3만원, 연결/설정은 무료)

## 업종별 특화

NexusPoint는 다음 업종에 특화된 템플릿과 경험을 보유하고 있습니다.

| 업종 | 주요 섹션 | 특화 기능 |
|------|----------|----------|
| 건설/시공업 | 시공 사례, 자격증, 문의 | 포트폴리오 갤러리 |
| 음식점/카페 | 메뉴, 위치, 예약 | 카카오맵 연동 |
| 뷰티/네일 | 서비스, 가격표, 예약 | 예약 폼 |
| 학원/교육 | 커리큘럼, 강사, 수강신청 | 온라인 문의 |

## FAQ

**Q. AI로 만든다는데 퀄리티가 괜찮나요?**
포트폴리오를 직접 확인해 주세요. AI는 도구일 뿐, 디자인 감각과 디테일은 사람이 잡습니다. AI 덕분에 시간과 비용이 절약되고, 그 혜택을 가격으로 돌려드립니다. → [PORTFOLIO.md](PORTFOLIO.md)

**Q. 쇼핑몰도 만들어주시나요?**
현재는 소개 사이트/랜딩페이지 전문입니다. 쇼핑몰은 별도 상담 부탁드려요.

**Q. 유지보수는 어떻게 되나요?**
Premium은 1개월 무료 유지보수 포함. 이후 월 5만원에 유지보수 계약 가능합니다. Basic/Standard는 납품 후 1주일 내 간단 수정만 지원합니다.
