# NexusPoint 에이전시 — 전체 개요

> 이 문서는 NexusPoint AI 웹개발 에이전시의 모든 것을 한 곳에서 파악할 수 있는 온보딩 가이드입니다.

---

## 1. 에이전시 소개

### 비전

> "AI 팀을 활용한 초저가 고품질 웹개발 서비스. 누구나 부담 없이 전문가 수준의 결과물을 받을 수 있다."

NexusPoint는 AI 기술의 발전으로 개발 원가가 거의 0에 수렴하는 현실을 활용, 기존 에이전시 대비 50~70% 저렴한 가격에 프리미엄 품질의 웹사이트를 제공합니다.

### 핵심 전략

**박리다매 (High Volume, Low Price)**

| 구분 | NexusPoint | 전통 외주 |
|------|-----------|----------|
| 납기 | 1~3일 | 2~4주 |
| 가격 | 30~200만원 | 300~1,000만원 |
| 작업 시간 | 24시간 | 근무시간만 |
| 수정 | 즉시 가능 | 추가 비용 |

### 팀 구성

```
Luchello (대표)
  ├─ 영업/소통: 클라이언트 미팅, 요구사항 청취, 최종 납품
  ├─ 사업 관리: 수금, 마케팅, 파트너십
  └─ 최종 승인: 납품 전 확인

Luveric (PM / 기술 총괄)
  ├─ 프로젝트 관리: 요구사항 → 태스크 분해 → 에이전트 배정
  ├─ 품질 관리: QA 루프 총괄, 최종 기술 검수
  │
  ├─ Builder (개발팀)
  │    ├─ spawn(opus): 핵심 코딩, 아키텍처
  │    └─ Claude Code CLI: 복잡한 리팩토링, 디버깅
  │
  ├─ QA (품질팀)
  │    ├─ 코드 리뷰, 빌드 검증
  │    └─ 접근성/SEO 감사
  │
  └─ Scout (조사/디자인팀)
       ├─ 시장 조사, 경쟁사 분석
       └─ 디자인 레퍼런스 수집
```

### AI 투명 정책

클라이언트에게 "AI 팀이 개발합니다"를 명시합니다. 이것이 오히려 경쟁력입니다. "그래서 이 가격에 이 속도가 가능합니다." 최종 품질은 QA 루프로 보장합니다.

## 2. 서비스 개요

서비스는 4개 등급으로 구성됩니다.

| 등급 | 가격 | 납기 | 대상 |
|------|------|------|------|
| Basic | 30~50만원 | 1~2일 | 랜딩페이지, 원페이지 사이트 |
| Standard | 50~100만원 | 2~5일 | 기업 홈페이지, 다중 섹션 |
| Premium | 100~200만원 | 5~14일 | 풀스택 웹앱, 관리자 패널 |
| Express | +50% | 24시간 이내 | Basic 한정 초급행 |

상세 내용: [SERVICES.md](SERVICES.md)

## 3. 기술 스택

**기본 스택** (Basic / Standard)
- Vite 6 + React 19 + Tailwind CSS v4
- lucide-react, Pretendard 폰트
- Vercel 호스팅

**고급 스택** (Premium)
- Next.js + Supabase (PostgreSQL + Auth)
- 토스페이먼츠 / 포트원 결제 연동

상세 내용: [TECH-SPEC.md](TECH-SPEC.md)

## 4. 프로젝트 진행 흐름

```
접수 → 견적 → 계약 → 개발 → QA → 납품 → 후속
```

1. **접수**: 고객 요구사항 수집 (업종, 페이지 수, 기능, 예산)
2. **견적**: 서비스 등급 + 추가 기능 산정
3. **계약**: 범위, 납기, 수정 횟수 명확화. 선금 50%
4. **개발**: 골든 템플릿 기반 커스터마이징
5. **QA**: 자동 감사 + 수동 체크리스트, 최소 2회전
6. **납품**: Vercel 배포, 도메인 연결, 사용 가이드 전달. 잔금 50%
7. **후속**: 만족도 확인, 유지보수 제안, 리뷰 요청

상세 내용: [PIPELINE.md](PIPELINE.md)

## 5. 품질 보증

QA는 최소 2회전 실시합니다.

- **1차 자동 감사**: 빌드, 링크, 반응형, 접근성, SEO, 성능, 보안
- **2차 수동 검수**: 콘텐츠, 디자인, 기능, 크로스 브라우저
- **문제 발견 시 3회전**까지 진행

철학: "납품 전에 우리가 고객보다 더 까다롭게 검수한다."

상세 내용: [QA.md](QA.md)

## 6. 비즈니스 모델

- **원가**: 서버 ~2만원/월 + Claude 구독 ~28만원/월 = **총 고정비 ~30만원/월**
- **수익**: 순이익률 90%+ (인건비 없음, AI가 개발)
- **목표**: Phase 1 기준 월 250~450만원 순이익

플랫폼 수수료 (견적에 반영 필수):
- 크몽: 20% / 숨고: 10~15% / 직접 영업: 0%

상세 내용: [BUSINESS-MODEL.md](BUSINESS-MODEL.md)

## 7. 포트폴리오

현재 제작 사이트 6개 + 에이전시 홈페이지 보유. 자동화 테스트 10건 실시, 9/10 통과, 평균 84초 완성.

## 8. 성장 로드맵

- **Phase 1** (1~3개월): 크몽 진입, 리뷰 5건+ 확보, 월 5~8건
- **Phase 2** (3~6개월): 직접 영업 확대, 유지보수 계약, 월 10~15건
- **Phase 3** (6~12개월): 템플릿 완성, 자동화 파이프라인, 월 20건+
