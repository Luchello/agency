|:---:|---|:---:|---|---|
| **1** | 🛒 **쇼핑몰 / 이커머스** | ~25% | 크몽, 위시켓, 숨고 | 자사몰(카페24/고도몰 커스텀), 입점형 오픈마켓, D2C 브랜드몰 |
| **2** | 🏢 **기업 홈페이지 / 랜딩페이지** | ~22% | 크몽, 숨고 | 소규모 사업장 소개 사이트, 원페이지 랜딩, 워드프레스/그누보드 기반 |

| **5** | 🔧 **기존 사이트 리뉴얼 / 유지보수** | ~15% | 크몽, 숨고, 위시켓 | 디자인 리뉴얼, 기능 추가, 레거시 마이그레이션, 반응형 전환 |

### 세부 트렌드 (2024-2025)
- **AI 기능 통합 의뢰 급증**: 챗봇, AI 추천, AI 검색 등을 기존 서비스에 붙이는 의뢰 증가
- **노코드/로코드 → 커스텀 전환**: Cafe24·아임웹으로 시작 후 한계를 느끼고 커스텀 개발 의뢰
- **MVP 개발 수요**: 스타트업의 투자 유치용 MVP 개발 의뢰가 위시켓에서 꾸준히 상위권

## 4. 납기 기대치

| 의뢰 유형 | 클라이언트 기대 납기 | 현실적 납기 | 비고 |
|---|---|---|---|
| **기업 홈페이지** | 1~2주 | 2~4주 | 템플릿 기반은 1~2주 가능 |
| **랜딩페이지 (1page)** | 3~7일 | 1~2주 | 크몽에서 빠른 납기 선호 |
| **쇼핑몰 (템플릿 커스텀)** | 2~3주 | 3~6주 | Cafe24/고도몰 기반 |
| **쇼핑몰 (풀 커스텀)** | 1~2개월 | 2~4개월 | 기획 포함 시 더 길어짐 |
| **플랫폼 서비스 MVP** | 1~2개월 | 2~4개월 | 기획 미비 시 +1~2개월 |
| **플랫폼 서비스 (풀)** | 3~4개월 | 4~8개월 | 앱 포함 시 6개월+ |
| **ERP/CRM** | 2~3개월 | 3~6개월+ | 규모에 따라 1년+ |
| **리뉴얼 / 유지보수** | 1~2주 | 1~4주 | 범위에 따라 크게 달라짐 |

### 📌 납기 관련 핵심 인사이트
- **클라이언트의 기대 납기는 현실의 50~70% 수준** — 실제보다 빠르게 기대하는 경향
- **기획 준비도가 납기를 좌우** — 위시켓 블로그에서도 "기획 없이 시작하면 납기 2배"라고 경고
- **크몽/숨고 의뢰**: "빠른 납기"가 중요한 선택 기준 → 소규모 작업 위주
- **위시켓 의뢰**: 매니저가 현실적 일정 컨설팅 → 기대치 조율이 비교적 잘 됨

## 5. 가장 경쟁력 있는(돈 되는) 의뢰 유형

### 🏆 수익성 TOP 3

#### 1위: ⚙️ ERP/CRM/업무 자동화 시스템
| 항목 | 내용 |
|---|---|
| **평균 프로젝트 단가** | 8,000만 ~ 1.6억원 |
| **왜 돈이 되나** | 높은 진입 장벽 (도메인 지식 필요), 경쟁자 적음, 장기 유지보수 계약 가능 |
| **필요 스킬** | 업무 프로세스 이해, 풀스택 개발, DB 설계 |
| **리스크** | 요구사항 변경 잦음, 장기 프로젝트로 현금흐름 불안정 |
| **추천 플랫폼** | 위시켓 |

#### 2위: 📱 플랫폼 서비스 (매칭/O2O/SaaS) MVP
| 항목 | 내용 |
|---|---|
| **평균 프로젝트 단가** | 1,000만 ~ 5,000만원 |
| **왜 돈이 되나** | 수요 꾸준함, 기술 난이도 중~상, 후속 작업(유지보수/확장) 높음 |
| **필요 스킬** | React/Next.js, Node.js/Python, 실시간 통신, 결제 연동 |
| **리스크** | 기획 미비 클라이언트 많음 → 스코프 크립 위험 |
| **추천 플랫폼** | 위시켓 |

#### 3위: 🛒 커스텀 이커머스 (D2C 브랜드몰)
| 항목 | 내용 |
|---|---|
| **평균 프로젝트 단가** | 500만 ~ 3,000만원 |
| **왜 돈이 되나** | 의뢰 빈도 최고, 반복 수주 용이, 템플릿화 가능 |
| **필요 스킬** | Cafe24/Shopify 커스텀, React, 결제 PG 연동 |
| **리스크** | 가격 경쟁 치열, 클라이언트의 빠른 납기 기대 |
| **추천 플랫폼** | 크몽, 위시켓 |

  │                             │
  │   기업 홈페이지 🏢           │ ← 고빈도, 저단가
  │   랜딩페이지 📄             │ ← 가격 경쟁 레드오션

### 🔥 2025-2026 주목할 블루오션
1. **AI 기능 통합 개발** — 기존 서비스에 AI 챗봇/추천/검색 붙이기 (위시켓에서 전환율 280% 상승 사례)
2. **노코드→커스텀 마이그레이션** — 아임웹/Cafe24 한계 도달 기업의 풀 커스텀 전환
3. **SaaS MVP 전문** — 스타트업 타겟, 빠른 MVP + 지분/수익 쉐어 모델
4. **레거시 시스템 현대화** — PHP/jQuery → React/Next.js 마이그레이션
