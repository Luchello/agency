## 가격이 낮아도 되는 이유
1. **인건비 0**: AI가 개발 → 인건비 없음
2. **속도**: 1-3일 완성 → 동시에 여러 프로젝트 가능
3. **재사용**: 업종별 템플릿 축적 → 갈수록 빨라짐
4. **고정비만**: 서버($15) + Claude($200) = 월 ~30만원

## 수익 시뮬레이션

### ⚠️ 수수료 고려
- 크몽: 매출의 20% 수수료 → 30만원 받으면 실수령 24만원
- 숨고: 매출의 10-15%
- 직접 영업: 수수료 0% (카카오톡/전화)
- 세금: 3.3% 원천징수 (프리랜서) 또는 10% 부가세 (사업자)

**전략**: 초기엔 크몽으로 리뷰 확보 → 점차 직접 영업 비중 높이기

## 업종별 템플릿 전략
반복 주문이 많은 업종별로 미리 틀을 만들어두면 개발 시간 대폭 단축:

1. **건설/시공업** — 이미 대성몰탈로 경험 있음 ⭐
2. **음식점/카페** — 메뉴, 위치, 예약
3. **미용/뷰티** — 서비스, 가격, 예약
4. **학원/교육** — 커리큘럼, 강사, 수강신청
5. **의료/치과** — 진료과목, 의료진, 예약
6. **쇼핑몰** — 상품 목록, 장바구니, 결제

각 템플릿 = 기본 디자인 + 컴포넌트 + 컬러 커스터마이징만 하면 됨

## 초기 전략 (1-3개월)
1. **크몽에 등록** — 최저가 포지셔닝 (30만원~)
2. **첫 3건 할인** — 20만원으로 진행, 리뷰 확보
3. **대성몰탈 사이트를 포트폴리오로** 활용
4. **빠른 납품**으로 차별화 — "1일 완성" 강조
5. **후기 마케팅** — 만족한 고객 리뷰 적극 활용
