# 대성몰탈 포트폴리오 사이트 v2 — Design Brief

## 사업 정보
- **업체명:** 대성몰탈
- **시공 분야:** 방통 (몰탈 바닥 수평) + 재물 (콘크리트 타설 후 표면 마감)
- **시공 지역:** 충청권 · 경기도권
- **전화:** 010-5535-7129
- **업무 시간:** 평일 08:00 ~ 18:00
- **경력:** 10+ 년
- **특기:** 500+ 현장 완료, 100% 책임 시공, 무료 견적

## 타겟 고객
- 건설사 현장소장, 인테리어 업체, 건물주
- 바닥 미장 시공이 필요한 충청/경기 지역 사업자

## 디자인 방향

### 핵심 키워드
**신뢰 · 견고 · 전문성 · 깔끔함**

### 컬러 팔레트
- **Primary:** 차콜/다크그레이 (#2D2D2D) — 콘크리트, 견고함
- **Accent:** 웜 앰버/골드 (#C4913E) — 전문성, 프리미엄
- **Background:** 오프화이트 (#F5F3F0) — 깨끗함, 밝음
- **Text:** 다크 (#1A1A1A)
- **Subtle:** 라이트그레이 (#E8E6E3) — 콘크리트 텍스처 느낌

### 타이포그래피
- 한국어: Pretendard (본문) + 볼드 웨이트로 임팩트
- 영문 숫자: Inter 또는 DM Sans
- 큰 숫자 강조 (10+년, 500+현장 등)

### 분위기
- **사진 중심** — 실제 현장 사진 19장 보유 (큰 장점!)
- 미니멀하지만 견고한 느낌
- 콘크리트/시멘트 텍스처를 은은하게 배경에
- 가짜 스톡 느낌 NO → 실제 사진만 사용
- 이모지 장식 NO

## 섹션 구성 (각각 다른 레이아웃!)

### 1. Hero — 풀스크린 사진 + 오버레이 텍스트
- 배경: 가장 임팩트 있는 현장 사진 (풀블리드)
- 오버레이: 어두운 그라디언트
- "대성몰탈" 큰 타이틀 + "바닥이 기초입니다" 서브카피
- CTA: "무료 견적 문의" 버튼 + 전화번호
- 숫자 뱃지: 10+년 / 500+현장 / 충청·경기

### 2. 소개 — 비대칭 2컬럼 (사진 + 텍스트)
- 왼쪽: 작업 중 사진 (오프셋, 약간 기울어진 프레임)
- 오른쪽: 대성몰탈 소개 텍스트
- "방통부터 재물까지, 바닥의 모든 것"
- 핵심 강점 3가지 (아이콘 + 짧은 설명)

### 3. 서비스 — 카드가 아닌 가로 스크롤 패널
- 방통 (몰탈 바닥 수평)
- 재물 (콘크리트 표면 마감)
- 각 서비스: 큰 사진 배경 + 설명 오버레이
- 좌우 스크롤 또는 탭 전환

### 4. 포트폴리오/시공사례 — 자유 배치 모자이크 그리드
- 실제 현장 사진 활용 (19장 중 선별)
- 다양한 크기의 사진 타일
- 호버 시 현장 정보 표시
- 라이트박스 (클릭 시 풀사이즈)
- 카테고리 필터: 방통 / 재물 / 전체

### 5. 시공 프로세스 — 수평 타임라인
- 5단계: 문의 → 현장방문 → 견적 → 시공 → 완료
- 수평 스크롤 또는 스텝 인디케이터
- 각 단계 아이콘 + 짧은 설명
- 세로가 아닌 가로 흐름 (차별화)

### 6. 장비 — 스태거 그리드 (큰/작은 교차)
- 보유 장비 사진
- 큰 타일 + 작은 타일 교차 배치
- 트럭, 호스, 바닥 장비 등

### 7. 숫자로 보는 대성몰탈 — 큰 숫자 + 카운터 애니메이션
- 10+ 년 경력
- 500+ 시공 현장
- 100% 책임 시공
- 충청·경기 시공 지역
- 다크 배경 + 골드 숫자 (임팩트)

### 8. 연락처 — 풀폭 다크 섹션
- 전화번호 크게 (클릭 시 바로 전화)
- 업무 시간
- 시공 가능 지역
- 간단한 문의 폼 (이름, 연락처, 시공 종류, 메시지)
- Formspree 연동

### 9. Footer — 미니멀
- 업체명, 전화, 주소
- 맨 위로 스크롤 버튼

## 기술 스택
- Vite + React + Tailwind CSS v4
- Pretendard 폰트
- IntersectionObserver 스크롤 애니메이션
- 반응형 (모바일 퍼스트)
- 라이트박스 (사진 클릭)
