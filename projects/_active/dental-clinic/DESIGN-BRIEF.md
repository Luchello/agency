# 치과 포트폴리오 — Design Brief

## 가상 브랜드
- **이름:** 미소플러스 치과 (MISO+ Dental)
- **컨셉:** "편안한 미소, 정확한 진료" — 강남 프리미엄 치과
- **위치:** 서울 강남구 역삼동
- **진료시간:** 평일 09:30–18:30 / 토 09:30–14:00 / 일·공휴일 휴진
- **연락처:** 02-555-2080 / @misoplus.dental
- **예약:** 전화 + 네이버 예약

## 컬러 팔레트
- Background: #F8FAFB (쿨 화이트)
- Surface: #FFFFFF
- Text: #1A2332 (다크 네이비)
- Text Secondary: #6B7B8D
- Accent: #2E7D8C (티얼 블루, 신뢰/청결)
- Accent Light: #E8F4F6
- Gold: #C4A265 (프리미엄 느낌)
- Dark Section BG: #1A2332
- Border: #E2E8ED

## 타이포그래피
- Display: **Playfair Display** (세리프)
- Body: **Noto Sans KR**

## 디자인 원칙
- **신뢰감** — 깔끔, 밝음, 의료적 청결감
- **프리미엄** — 골드 액센트, 세리프 타이포
- **안심** — 따뜻한 카피, 의료진 소개
- 이모지 장식 절대 금지

## 섹션 구성 (8개)

### 1. Hero — 풀스크린 + 오버레이
- "편안한 미소, 정확한 진료"
- "강남역 3번 출구 · 20년 경력"
- CTA: "진료 예약" + "진료 안내"
- 하단: 소수점 없는 숫자 뱃지 (20년 경력 / 5만+ 환자 / 4.9 평점)

### 2. Services — 아이콘 카드 그리드 (6개)
- 임플란트, 교정, 심미보철, 충치치료, 스케일링, 미백
- 각 카드: 아이콘 + 제목 + 짧은 설명
- 2행 3열 그리드 (모바일 1열)

### 3. About — 2컬럼 (의료진 이미지 + 소개)
- 원장 소개 (이름, 학력, 경력)
- "환자 한 분 한 분에게 최선의 진료를"
- 자격증/학회 로고 배열

### 4. Before/After — 가로 스크롤 snap
- 치료 전후 비교 카드 3-4개
- 각 카드: Before → After 슬라이더 느낌 (CSS)

### 5. Facility — 풀폭 이미지 갤러리
- 진료실, 대기실, 장비 사진 4장
- snap 스크롤

### 6. Reviews — 다크 배경 + 리뷰 카드
- 환자 리뷰 3개
- 별점 + 텍스트 + 치료 종류 태그

### 7. Appointment — 티얼 배경 + CTA
- "지금 예약하세요"
- 전화번호 (큰 글씨)
- 네이버 예약 버튼
- 진료시간 안내

### 8. Footer — 다크 네이비
- 로고 + 주소 + 전화 + SNS + 사업자 정보

## 기술
- Vite + React + Tailwind CSS v4
- 모바일 퍼스트 반응형
