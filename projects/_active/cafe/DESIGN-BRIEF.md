# 카페 포트폴리오 — Design Brief

## 가상 브랜드
- **이름:** 온도 (ONDO)
- **컨셉:** "적정 온도의 커피와 시간" — 슬로우 로스팅, 핸드드립 전문 카페
- **위치:** 서울 성수동
- **영업시간:** 매일 09:00–22:00
- **연락처:** 02-1234-5678 / @ondo.coffee

## 레퍼런스 분석
- **프리츠(Fritz):** 미니멀, 레트로 타이포, 여백 많음, 커피클럽 구독 모델
- **블루보틀:** 극도의 미니멀, 단색 로고, 여백이 곧 브랜드
- **어니언:** 공간 사진 중심, 건축적 느낌, 무채색 + 자연광
- **테라로사:** 원두 산지 스토리텔링, 갈색/크래프트 톤

## 컬러 팔레트
- Background: #FAF8F5 (웜 크림)
- Surface: #FFFFFF
- Text: #2C2420 (다크 브라운)
- Text Secondary: #8B7D6B
- Accent: #C5956B (카라멜 골드)
- Dark Section BG: #2C2420
- Border: #E8E2DA

## 타이포그래피
- Display: **Playfair Display** (세리프)
- Body: **Noto Sans KR**
- Accent/Label: DM Sans (영문 라벨)

## 디자인 원칙
1. **여백 = 고급** (블루보틀/프리츠급 여백)
2. **사진이 왕** (커피, 공간, 원두 클로즈업)
3. **따뜻한 톤** (차가운 미니멀이 아닌, 따뜻한 미니멀)
4. **세리프 타이포** (Playfair Display로 크래프트 감성)
5. **이모지 장식 절대 금지**
6. **각 섹션 레이아웃 반복 금지**

## 섹션 구성 (8개, 각각 다른 레이아웃)

### 1. Hero — 풀스크린 이미지
- 커피 내리는 장면 or 카페 인테리어
- 그라데이션 오버레이 (다크 브라운 → 투명)
- Playfair Display "적정 온도의 커피와 시간"
- "Since 2019 · Seoul Seongsu"
- CTA: "메뉴 보기" + "방문 예약"

### 2. Philosophy — 2컬럼 (텍스트 + 이미지)
- "한 잔의 커피가 완성되기까지"
- 슬로우 로스팅, 싱글 오리진, 핸드드립 철학
- 오른쪽: 원두/로스팅 이미지

### 3. Menu Highlights — 비대칭 카드 그리드
- 시그니처 메뉴 4-6개
- 이름 + 가격 + 짧은 설명
- 카드마다 다른 크기 (벤토 느낌)
- 아메리카노 5,500 / 라떼 6,500 / 드립커피 7,000 / 크로플 6,000

### 4. Space — 풀폭 가로 스크롤 갤러리
- 카페 공간 사진 4-5장
- snap 스크롤
- 캡션 오버레이

### 5. Origin Story — 다크 배경 섹션
- 원두 산지 이야기
- "에티오피아 예가체프 → 콜롬비아 우일라"
- 지도 느낌의 시각 요소 (CSS로)
- 숫자 카운터: 3개국 산지 / 주 3회 로스팅 / 12종 블렌드

### 6. Reviews — 가로 스크롤 리뷰 카드
- 네이버/카카오 리뷰 발췌
- 별점 + 텍스트 + 작성자

### 7. Visit — 2컬럼 (지도 + 정보)
- 지도 플레이스홀더 (CSS 그라데이션)
- 주소, 전화, 영업시간, 주차
- 네이버 지도 링크

### 8. Footer — 다크 브라운
- 로고 + SNS + 사업자 정보
- 인스타그램 링크
