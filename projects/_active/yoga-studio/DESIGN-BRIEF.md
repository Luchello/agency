## 레퍼런스 분석

### 디자인 패턴 (Awwwards + Behance 인기 사이트)
1. **Radiant Hot Yoga** (Awwwards) — 풀스크린 비디오 히어로, 따뜻한 톤, 대형 타이포
2. **Yoga Plus** (Awwwards) — 미니멀 그리드, 클래스 카드, 깔끔한 예약 플로우
3. **CASA CORPO** (Behance 373♥) — 유기적 곡선, 라이트 베이지 배경, 세리프 타이포, 여백 활용 극대화
4. **Slow Soul Meditation** (Behance 332♥) — 명상적 톤, 파스텔 그라데이션, 부드러운 애니메이션
5. **Balance Yoga Studio** (Behance 176♥) — 좌우 비대칭 레이아웃, 인스트럭터 소개 강조
6. **Landing page for yoga studio** (Behance 5,554♥ — 최고 인기!) — 유기적 블롭 도형, 라벤더+크림 팔레트, 플로팅 이미지

### 추출한 핵심 디자인 원칙
- **호흡하는 레이아웃**: 극도의 여백 (요가의 "여백 = 고요함" 메타포)
- **유기적 곡선**: blob/wave 도형, 둥근 모서리 (직선 최소화)
- **풀스크린 히어로**: 요가 포즈 + 빛이 들어오는 스튜디오 사진
- **대형 세리프 타이포**: 우아하고 고요한 느낌 (Playfair Display or Cormorant)
- **파스텔 그라데이션**: 라벤더→크림 or 세이지→화이트 자연스러운 전환
- **플로팅/오프셋 이미지**: 그리드에 갇히지 않은 자유로운 이미지 배치
- **인스트럭터 카드**: 원형 사진 + 이름 + 전문분야
- **클래스 시간표**: 요일별 그리드 or 탭 UI

## 섹션 구성
1. **Hero** — 풀스크린 요가 포즈 이미지 + 세리프 헤드라인 + blob 오버레이
2. **Philosophy** — "균형, 호흡, 성장" 3개 가치 + 아이콘 (텍스트+이미지 반반, 오프셋)
3. **Classes** — 클래스 카드 (하타/빈야사/아쉬탕가/핫요가) + 각각 다른 배경색 그라데이션
4. **Instructor** — 원형 사진 + 이름 + 자격 + 짧은 소개
5. **Schedule** — 요일별 시간표 그리드 (파스텔 셀)
6. **Gallery** — 오프셋 그리드 (정렬되지 않은 자유 배치) + 호버 줌
7. **Testimonials** — 큰 따옴표 + 후기 + 회원 이름
8. **Location** — 지도 + 시설 안내 + 연락처
9. **Footer** — 미니멀, SNS 링크
