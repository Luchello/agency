# 네일샵 포트폴리오 — Design Brief

## 가상 브랜드
- **이름:** 라뮤르 (LA MURE)
- **컨셉:** "손끝에 담은 프렌치 감성" — 프리미엄 네일 아트 & 케어
- **위치:** 서울 압구정동
- **영업시간:** 화-일 11:00–21:00 (월요일 휴무)
- **연락처:** 02-567-8901 / @lamure.nail
- **예약:** 네이버 예약 연동

## 레퍼런스 스타일
- 프렌치 미니멀 뷰티 (Glossier + Aesop 감성)
- 네일 사진이 주인공 — 아트워크 클로즈업
- 부드러운 핑크/누드 톤 + 깔끔한 여백

## 컬러 팔레트
- Background: #FDF8F6 (소프트 핑크 크림)
- Surface: #FFFFFF
- Text: #2D2024 (다크 모브)
- Text Secondary: #9B8B91
- Accent: #D4A0A0 (더스티 로즈)
- Accent Secondary: #B8946B (골드 누드)
- Dark Section BG: #2D2024
- Border: #EDE5E2

## 타이포그래피
- Display: **Playfair Display** (세리프)
- Body: **Noto Sans KR**

## 섹션 구성 (8개)

### 1. Hero — 풀스크린 네일아트 클로즈업
- "손끝에 담은 프렌치 감성"
- "Apgujeong · Since 2021"
- CTA: "예약하기" + "아트 갤러리"

### 2. About — 2컬럼 (큰 이미지 + 텍스트)
- 브랜드 소개
- "섬세한 손길, 프리미엄 제품, 당신만의 디자인"
- 자격증/경력 뱃지

### 3. Services — 서비스 카드 그리드 (3개)
- 젤네일 (기본 59,000~ / 아트 79,000~)
- 패디큐어 (49,000~)
- 네일케어 (39,000~)
- 각 카드에 아이콘 + 설명 + 가격

### 4. Gallery — 마소닉/벤토 그리드
- 네일아트 작품 사진 6-8장
- 호버 시 확대 효과
- 카테고리: 프렌치 / 시즌 / 웨딩 / 아트

### 5. Process — 수평 타임라인 (4단계)
- 상담 → 디자인 선택 → 시술 → 애프터케어
- 각 단계 아이콘 + 설명

### 6. Reviews — 크림 배경 + 리뷰 카드 3개
- 네이버/카카오 리뷰 발췌
- 별점 + 텍스트

### 7. Booking — 다크 배경 + CTA
- "지금 예약하세요"
- 네이버 예약 버튼 (큰 CTA)
- 전화번호
- 운영 시간

### 8. Footer — 다크 모브
- 로고 + SNS + 사업자 정보

## 이미지 (CSS 그라데이션 플레이스홀더)
- 더스티 로즈 + 골드 누드 톤의 따뜻한 그라데이션

## 기술
- Vite + React + Tailwind CSS v4
- 모바일 퍼스트 반응형
