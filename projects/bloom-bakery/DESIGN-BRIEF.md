# Bloom Bakery — 디자인 브리프

- **업체명:** 블룸 베이커리 (Bloom Bakery)
- **위치:** 서울 성수동
- **컨셉:** 매일 아침 정성을 굽는 수제 빵 & 케이크
- **영업시간:** 08:00~21:00, 월요일 휴무
- **전화:** 02-1234-5678
- **인스타:** @bloom_bakery_ss

## 레퍼런스 분석 (Awwwards + Behance 인기 베이커리 사이트)

### 디자인 패턴 (장점만 추출)
1. **Bernice Bakery** — 플레이풀한 마퀴 텍스트 ("Bestsellers" 무한 스크롤), 감성 카피 ("Good food should comfort and nourish the soul"), Shopify 스타일 제품 카드
2. **Wildwood Bakery** (8.5+/10) — 내추럴/오가닉 톤, 풀스크린 이미지, 자연광 사진 중심
3. **Fritz Coffee (한국)** — 극도로 미니멀, 매장 위치 중심, 한영 병기 타이포, 여백 활용
4. **Heaven Bites** (Behance 1,002♥) — 대형 히어로 이미지, 고급 컬러 팔레트, 섹션별 차별화
5. **Scandibakes** — 스칸디나비안 클린, 그리드 기반, 화이트 스페이스 극대화
6. **Bammi** — 모던 유기적 곡선, 유니크 섹션 전환, 브랜드 일관성

### 추출한 핵심 디자인 원칙
- **풀스크린 히어로**: 빵/케이크 클로즈업 이미지가 화면 전체를 채움
- **마퀴 텍스트**: "갓 구운 빵" 같은 문구가 좌→우 무한 스크롤 (Bernice 스타일)
- **화이트 스페이스**: Fritz/Scandibakes처럼 여백을 충분히 (빵의 따뜻함이 숨쉬게)
- **대형 타이포**: 헤드라인은 크고 임팩트있게, 세리프+산세리프 믹스
- **자연광 톤**: 따뜻한 크림/베이지 배경 + 갈색/골드 액센트
- **제품 중심 갤러리**: 격자가 아닌 자유 배치 or 마소닉 그리드
- **카피 톤**: 따뜻하고 감성적 ("매일 아침, 정성을 굽습니다")
- **스크롤 인터랙션**: 섹션 진입 시 자연스러운 페이드인 + 패럴랙스

## 색상
- primary: `#C4733B` (따뜻한 시나몬)
- accent: `#8B5E3C` (다크 초콜릿)
- bg: `#FDF8F3` (크림)
- text: `#3D2B1F` (에스프레소)

## 섹션 구성 (권장, 변형 가능)
1. **Hero** — 풀스크린 빵 이미지 + 헤드라인 + CTA
2. **Marquee** — "갓 구운 빵 · 수제 케이크 · 브런치 세트 · 커피 & 음료" 무한 스크롤
3. **Story** — "2020년 성수동 골목에서 시작한..." 스토리 (이미지+텍스트 반반)
4. **Menu/Products** — 시그니처 빵, 케이크, 브런치 카드
5. **Gallery** — 매장/빵 사진 자유 배치
6. **Reviews** — 고객 후기 3개
7. **Location & Hours** — 지도 + 영업시간 + 연락처
8. **Footer** — SNS + 저작권

## 기술 스택
- Vite + React + Tailwind CSS v4
- 싱글 페이지, 반응형
- 이미지는 ChatGPT로 생성 후 WebP 변환

## 절대 하지 말 것
- ❌ 이모지를 장식으로 사용
- ❌ 스톡 사진 느낌
- ❌ 기존 골든 템플릿과 같은 구조
- ❌ 섹션마다 동일한 카드 레이아웃 반복
