# 🔄 프로젝트 파이프라인

## 전체 흐름
```
접수 → 견적 → 계약 → 개발 → QA → 납품 → 후속
```

---

## 1. 📋 접수 (Intake)
**담당: Luchello**

클라이언트로부터 수집할 정보:
- [ ] 업체/개인 이름
- [ ] 연락처 (카카오톡/전화)
- [ ] 원하는 사이트 종류 (랜딩/홈페이지/쇼핑몰/기타)
- [ ] 페이지 수 (대략)
- [ ] 레퍼런스 사이트 (있으면)

- [ ] 콘텐츠 준비 상태 (텍스트/이미지/로고)
- [ ] 희망 납기
- [ ] 예산 범위

- 크몽/숨고: 플랫폼 메시지

**접수 후 Luchello → Luveric에게 전달:**

## 2. 💰 견적 (Estimate)
**담당: Luveric (자동 산정) → Luchello (확인/전달)**

### 자동 견적 공식
```
기본가 = 서비스 등급 기본료
+ (페이지 수 - 기본 포함 수) × 5만원
+ 추가 기능별 가격
+ Express 옵션 (해당 시)
```

### 추가 기능 가격표
| 기능 | 가격 |
|------|------|
| 문의 폼 (이메일 연동) | +5만원 |
| 카카오톡 채팅 연동 | +3만원 |
| 다국어 (2개 언어) | +15만원 |
| 블로그/뉴스 섹션 | +10만원 |
| 쇼핑몰 (프론트만) | +30만원 |
| 관리자 패널 | +50만원 |
| 결제 연동 | +40만원 |
| SEO 심화 (네이버 최적화) | +10만원 |
| 도메인 연결 + SSL | +5만원 |
| 월 유지보수 | 5-10만원/월 |

### 견적서 템플릿 → `templates/estimate.md`

## 4. 🛠️ 개발 (Development)
**담당: Luveric → Builder 팀**

### 개발 프로세스
```
4-1. 프로젝트 셋업
     ├─ 레포 생성 (GitHub)
     ├─ 기술 스택 결정 (기본: Vite + React + Tailwind)
     └─ 디렉토리 구조 세팅

4-2. 디자인 결정
     ├─ 컬러 팔레트 (고객 브랜드 기반)
     ├─ 레이아웃 구조
     └─ 레퍼런스 참고

4-3. 개발 (자가개선 루프 적용)
     ├─ Builder(spawn/opus)에게 위임
     ├─ 3-5개 태스크 단위로 진행
     ├─ 매 라운드 후 검증
     └─ lessons.md 교훈 반영

4-4. 콘텐츠 적용
     ├─ 고객 제공 텍스트/이미지 삽입
     ├─ 이미지 최적화 (WebP 변환)
     └─ SEO 메타태그 설정
```

### 기술 스택 (기본)
- **프론트엔드**: Vite + React + Tailwind CSS
- **호스팅**: Vercel (무료) 또는 GitHub Pages
- **도메인**: 고객 기존 도메인 또는 신규 구매
- **폼**: Web3Forms / FormSubmit (무료 백엔드)
- **분석**: Google Analytics (무료)

### 고급 스택 (Premium)
- **백엔드**: Next.js API Routes / Supabase
- **DB**: Supabase (PostgreSQL) / Firebase
- **인증**: Supabase Auth / NextAuth
- **결제**: 토스페이먼츠 / 포트원

## 5. ✅ QA (Quality Assurance)
**담당: Luveric → QA 팀**

→ 상세: `qa/QA.md`

QA는 최소 2회전 실시:
1. **자동 감사**: 빌드, 링크, 접근성, SEO, 성능
2. **수동 검수**: 체크리스트 기반 확인
3. **수정 후 재검수**: 문제 발견 시 수정 → 재확인

---

## 6. 📦 납품 (Delivery)
**담당: Luveric (배포) → Luchello (고객 전달)**

### 납품물
- [ ] 라이브 URL (Vercel/GitHub Pages)
- [ ] GitHub 레포 접근권 (고객 요청 시)
- [ ] 도메인 연결 (해당 시)
- [ ] 사용 가이드 (간단한 수정 방법)
- [ ] 소스 코드 (요청 시)

### 납품 후
- 고객 확인 기간: 3일
- 수정 요청 → 계약된 횟수 내 무료 수정
- 확인 완료 → 잔금 수령

---

## 7. 🔄 후속 (Follow-up)
**담당: Luchello**

- 1주 후 만족도 확인
- 유지보수 계약 제안
- 리뷰/후기 요청 (크몽/숨고 평점)
- 추가 작업 제안 (SEO 개선, 신규 페이지 등)
